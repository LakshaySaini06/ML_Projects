# ML_Projects

# 1. Rag based Teaching Assistant
Built an AI assistant that answers user questions from course videos using Retrieval-Augmented Generation (RAG). Converted videos to transcripts, generated embeddings, and retrieved relevant video timestamps using cosine similarity. Integrated LLMs to generate contextual, human-like responses.

# 2.Price Prediction Model
Designed a machine learning regression pipeline with preprocessing and model training using scikit-learn. Applied feature engineering, stratified sampling, and cross-validation to evaluate model performance. Achieved reliable house price predictions using Random Forest and regression models.

# 3. Coders of Gurgaon
Analyzed social media data of Indian tech professionals and communities using Python. Performed data cleaning and exploratory analysis to study engagement patterns and profile types. Extracted insights from JSON and text-based datasets to understand tech branding trends.
